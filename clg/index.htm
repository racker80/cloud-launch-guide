The Web Server

Getting your Cloud Server configured correctly is the goal of this walkthrough. In this section, we show you everything you need to do to make sure this web server is 100% WordPress ready.

Prepare for Scale

This exercise will demonstrate how the Rackspace Cloud infrastructure, when properly configured, can give you the ability to scale your site to meet demand. We’ll be doing what’s called “Horizontal Scale”. Horizontal scaling is the practice of using multiple servers to scale a configuration. Needless to say, you are going to have a very robust cloud configuration.

Checklist

Before we begin, let’s take a moment to look over what we will accomplish in this server build out:

  • Configure the Apache Web Server.
  • Provide Permalinks for clean URL Structures.
  • Create an FTP User for Uploading Files.
  • Create a Load Balanced Multinode configuration.
  • Configure user permissions for Apache and your FTP User.
  • Provide clean URL structures based on Permalinks using .htaccess.
  • Allow you to download and install WordPress plugins and Themes.
  • Create an image of your server for future Scaling.
  • Configure Replication and Proxy Software to support your MultiServer Configuration.
